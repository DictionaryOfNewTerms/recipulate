## Persona Template

When creating personas, include:

### Jobs To Be Done
- **Main Job**: What the user is trying to accomplish
- **Related Jobs**: Adjacent tasks they perform
- **Emotional Jobs**: How they want to feel

### Progress Outcomes
- What "better" looks like for this user
- Measurable success criteria

### Pain Points
- Current frustrations with existing solutions
- Barriers to progress

### Context
- When/where they encounter the problem
- Tools they currently use
